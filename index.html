Application Overview

ScrumptyDumpty is in the process of developing a tool which will aid in the shopping needs of its users. This application will be utilized in the form of a webpage which will allow its users to create an account, create labeled shopping lists with products, create different groups of shoppers, and view nearby locations to shop.

Technology Overview

As part of our efforts, we are utilizing a wide variety of other applications and services. These include using GitHub repositories and organizations as our production environment. The front end of our application will be created using HTML 5, Tailwind CSS, and JavaScript for functionality. MySQL and Node.js will be utilized on the back-end to store user account information and provide communication between the client and the server. In addition, Microsoft Azure will be the platform on which our web application will be hosted.

Feature List

  • Create a personal account
  • Create personal and shared shopping lists
  • Invite friends and family to groups to shop together
  • Locate nearby stores on a map

Future Feature List

  • Availability as a mobile application for both IOS and Android

Basic Technical Features

  • Utilizes Google Maps to provide an interactive map
  • CSS and HTML creates a sleek design of the webpage
  • Hosted as a web application on Azure

Advanced Technical Features

  • MySQL stores user account information in a database
  • Node.js retrieves required information from the server
